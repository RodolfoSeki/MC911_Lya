List of tests the compiler check

1 SYNONYM
  set list of synonyms of same type in the same declaration
  set synonyms of different type in the same declaration

2 TYPE
  create TYPE with list of modes

3 RECURSION
  recursive function

4 POINTERS
  pointer declaration
  function with pointer argument
  pointer operation
  assign NULL to pointer
  assign to pointer function that returns address
  
5 MATRIX
  matrix operations

6 CONDITIONAL
  conditional operations in if statement (>=, &&..)
